MatDialog.js is a JavaScript library for MatereializeCss which provides you functionalities to create dialog boxes without having to worry about creating, managing or removing any of the required DOM elements or JS event handlers.


Take a look at code for showing alert:
var MyDialog=new MatDialog(); MyDialog.alert( 'Hey I am Alert');

or if you are in hurry

new MatDialog().alert( 'Hey I am Alert');


It exposes four api -
1. Alert see documentation
var MyDialog=new MatDialog(); MyDialog.alert( 'Hey I am Alert',function(){ });
2. Confirm see documentation
//Assumimg we have already created the object. MyDialog.confirm( 'This is a confirm dialog',function(result){ });
3. Prompt see documentation
//Assumimg we have already created the object. MyDialog.prompt( 'This is a prompt dialog',function(result){ });
4. Create - By using this you can create your own customize dialog see documentation
//Assumimg we have already created the object. MyDialog.create({ Content:{ Text:'I am customized dialog box' } });
Default Alert
var MyDialog=new MatDialog(); MyDialog.alert( 'Hey I am Alert',function(){ });
Default Alert with callback
MyDialog.alert( 'Hey I am Alert with callback',function(){ });
Alert with customized button
MyDialog.alert( { Text:'Hey I am Alert with callback', Button:{ Label:'Ok great !', Class:'red' } }, function(){ } );
More Option :
ExecuteAfter - This will be called after the modal creation.
ExecuteBefore - This will be called before modal creation.
MyDialog.alert( { Text:'Hey I am Alert with callback', Button:{ Label:'Ok great !', Class:'red' }, ExecuteBefore:function(){ } }, function(){ } );
Default Prompt
var MyDialog=new MatDialog(); MyDialog.confirm( 'This is a confirm dialog',function(){ });
Confirm with customized button
MyDialog.confirm( { Text:'Hey I am Confirm Dialog', Buttons:{ Ok:{ Label:'Ok cool !', Class:'green' }, Cancel:{ Label:'Not cool !', Class:'red' } } }, function(){ } );
More Option :
ExecuteAfter - This will be called after the modal creation.
ExecuteBefore - This will be called before modal creation.
