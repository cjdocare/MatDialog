Create takes a javascript object, which should have format like this - 
    {
        Title: {
            Label: string,
            ShowClose: boolean
        },
        Content: {
            Label: string,
            Class: string
        },
        ButtonType: string
        Buttons: [{
            Label: string,
            Class: string,
            Value: string
        }],
        ExecuteAfter: Function,
        ExecuteBefore: Function
    }
Simple Customize Dialog
//Assumimg we have already created the object. MyDialog.create({ Content:{ Label:'I am customized dialog box', Class:'#cddc39 lime', } });
Customize Dialog With Title
MyDialog.create({ Content:{ Label:'I am customized dialog box', Class:'#cddc39 lime', }, Title:{ Label:'Hi', ShowClose:true, } });
Customize Dialog With Buttons
MyDialog.create({ Content:{ Label:'I am customized dialog box', Class:'#cddc39 lime', }, Buttons:[{ Label:'Button1', Class:'#388e3c green darken-2', }, { Label:'Button2', Class:'#00b8d4 cyan accent-4', }, { Label:'Button3', Class:'#ad1457 pink darken-3', }] }, function(result){ //callback } );
Note : you can also mention ButtonType: Alert or Prompt to get the already defined buttons.
