Create takes a javascript object, which should have format like this - 
    {
        Title: {
            Text: string,
            ShowClose: boolean
        },
        Content: {
            Text: string,
            Class: string
        },
        ButtonType: string
        Buttons: [{
            Label: string,
            Class: string,
            Value: string
        }],
        ExecuteAfter: Function,
        ExecuteBefore: Function
    }
Simple Customize Dialog
//Assumimg we have already created the object. MyDialog.create({ Content:{ Text:'I am customized dialog box', Class:'#cddc39 lime', } });
Prompt with input Type : text, date, number, email, password
MyDialog.prompt( { Text:'Hey I am prompt Dialog', Input:{ Type:'password', } }, function(result){ //callback } );
Prompt with input Type : select,radio,checkbox
MyDialog.prompt( { Text:'Things you care about?', Input:{ Type:'check', Values: [ { Text:'Education', Value:'ed' }, { Text:'Poverty', Value:'po' }, { Text:'Religion', Value:'re' } ] } }, function(result){ //callback } );
Prompt with customized button
MyDialog.prompt( { Text:'Hey I am prompt Dialog', Buttons:{ Ok:{ Label:'Ok cool !', // you can also add icon Class:'green' }, Cancel:{ Label:'Not cool !', Class:'red' } } }, function(result){ //callback } );
More Option :
ExecuteAfter - This will be called after the modal creation.
ExecuteBefore - This will be called before modal creation.
MyDialog.prompt( { Text:'Hey I am confirm', Button:{ Label:'Ok great !', Class:'red' }, ExecuteAfter:function(){ } }, function(result){ //callback } );
