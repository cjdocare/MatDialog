Default Confirm
var MyDialog=new MatDialog(); MyDialog.confirm( 'This is a confirm dialog',function(result){ //called });
Confirm with customized button
MyDialog.confirm( { Text:'Hey I am Confirm Dialog', Buttons:{ Ok:{ Label:'Ok cool !', // you can also add icon Class:'green' }, Cancel:{ Label:'Not cool !', Class:'red' } } }, function(result){ //callBack } );
More Option :
ExecuteAfter - This will be called after the modal creation.
ExecuteBefore - This will be called before modal creation.
MyDialog.confirm( { Text:'Hey I am confirm', Button:{ Label:'Ok great !', Class:'red' }, ExecuteAfter:function(){ } }, function(result){ //callBack } );
