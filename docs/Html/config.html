This is the default configuration. 
    {
            Dismissible: true, // Modal can be dismissed by clicking outside of the modal
            EndingTop: '2%', // Ending top style attribute
            InDuration: 300, // Transition in duration
            OutDuration: 200, // Transition out duration
            Opacity: 0.5, // Opacity of modal background
            StartingTop: '2%' // Starting top style attribute
    }

You can pass this json object to change the config. Check out the below example.

Using Constructor
var MyDialog=new MatDialog({ Dismissible: true, EndingTop: '5%', InDuration: 500, OutDuration: 1000, Opacity: 0.1, StartingTop: '10%' }); MyDialog.alert('config dialog');
Using setModalConfig - you can use this to change config at any time.
var MyDialog=new MatDialog(); MyDialog.setModalConfig({ Dismissible: false, EndingTop: '5%', InDuration: 500, OutDuration: 1000, Opacity: 0.1, StartingTop: '10%' }); MyDialog.alert('config dialog');
